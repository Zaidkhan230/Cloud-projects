AWS VPC Networking Architecture Lab

Project Overview
This project was created to understand the basics of AWS cloud networking and how networking resources work inside a cloud environment.
In this lab, I created a custom VPC architecture using public and private subnets, route tables, and an Internet Gateway. The project helped me understand how cloud resources communicate and how internet connectivity is controlled inside AWS.

The setup was built using beginner-level AWS networking concepts similar to real-world cloud environments.
What I Built
  - Created a custom VPC
  - Configured public and private subnets
  - Attached Internet Gateway
  - Created Route Tables
  - Associated subnets with route tables
  - Configured internet access for public subnets

 AWS Services & Concepts Used
  - Amazon VPC
  - Public Subnets
  - Private Subnets
  - CIDR Block Configuration
  - Route Tables
  - Internet Gateway
  - Subnet Association
  - AWS Networking Basics
